# Compose for Wear OS サンプル
Compose for Wear OSのサンプル実装アプリです。スタンドアロンで動くのでスマホや通信環境は必要ありません（デバッグのためには必要です）。

<img src="docs/app.gif" width=300 />

# 機能リスト

- button
- card
- chip
- dialog
- horologist
- list
- navigation
- pager
- picker
